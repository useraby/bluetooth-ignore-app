## Release 发布页文案

下面这版适合直接贴到 GitHub Release 正文：

```md
## 蓝牙设备一键断开忽略 v1.0.0

这是一个面向 macOS 的轻量工具，主要用于蓝牙串口调试场景下，对指定蓝牙设备执行“断开 + 忽略（取消配对）”操作。

### 为什么做这个工具

在 macOS 升级后，某些第三方蓝牙串口工具与系统兼容性会变差。对于网络设备调试场景，尤其是华为交换机等设备联调流程，连接过目标蓝牙设备后，往往还需要再次手动忽略该设备，否则下次无法继续正常使用串口工具。

这个 app 的目标，就是把这类重复操作压缩成一次双击。

### 本版本特性

- 首次运行可手动选择需要处理的蓝牙设备
- 支持“记住该设备”，后续双击直接执行
- 未记住设备时，每次运行都会重新选择
- 启动时按住 `Option` 键可强制重新选择设备
- 优先使用系统通知反馈执行结果
- 使用原生 macOS app 形式封装，便于直接双击使用

### 依赖

运行前请先安装：

```bash
brew install blueutil
```

### 适合谁

- 蓝牙串口调试用户
- 网络工程师
- 需要频繁断开并忽略固定蓝牙设备的 macOS 用户
```
